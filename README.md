**<h1>Snowballer</h1>**
This is a game where you fight "Snow Monsters" by throwing snowballs at them,
Your goal is to keep your "Base" safe and don't let any monsters near.
**<h1>Keybinds</h1>**
> **W -> Move Forward**
> **A -> Move Left**
> **S -> Move Backwards**
> **D -> Move Right**
> **Space -> Jump**
> **LMB -> Throw Snowball**
> **ESC -> Quit**
